## 🎯 Key Concepts Quick Guide

### Spacing Abbreviations
```
VS  = Vertical Spacing (gap between rows)
HS  = Horizontal Spacing (gap between columns)
MS  = Margin/Spacing (padding all sides)
WS  = Width Scale (responsive width)
HeS = Height Scale (responsive height)
```

### Spacing Scale
```
xs (4px)   → minimal
sm (8px)   → small
md (12px)  → medium
lg (16px)  → large (most used)
xl (20px)  → extra large
2xl-4xl    → progressively larger
```

### Color System
```
Primary:   #FF5864 (Tinder Red)
Secondary: #0099FF (Tinder Blue)
Tertiary:  #FAB938 (Tinder Gold)
Like:      #FF5864 (Red)
Pass:      #9CA3AF (Gray)
SuperLike: #FAB938 (Gold)
Match:     #3FD4B4 (Green)
```

### Architecture Layers
```
Screens       (Full pages)
  ↓
Organisms     (Complex sections)
  ↓
Molecules     (Component combinations)
  ↓
Atoms         (Basic elements)
  ↓
Styling       (Theme, colors, spacing)
```
